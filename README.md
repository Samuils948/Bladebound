# Bladebound

## About

Bladebound is a small 2D action RPG survival game made in Godot 4. Fight waves of bats, earn points, and see how long you can stay alive. The game uses retro-style pixel art, simple combat, and gets harder as more enemies appear over time.

## Features

* Retro pixel-art graphics
* Character and enemy animations
* Background music during gameplay
* Sound effects for attacks, rolls, hits, and other actions
* Waves of enemies
* Score system
* Survival timer
* Game Over screen
* Pause menu
* Increasing difficulty

## Controls

| Key           | Action       |
| ------------- | ------------ |
| W / A / S / D | Move         |
| J or X        | Attack       |
| K or Z        | Roll / Dodge |
| Esc           | Pause        |

## How to Play

Defeat bats to earn points and survive for as long as possible. More enemies will spawn as time goes on, making the game harder. Try to beat your best score and survive longer each run.

## Built With
